Provides some examples of using the egalito shell (version 2). Specifically,
invokes etshell2 in (egalito/app) with commands provided in the file provided
as input.

shell.in is the default input file. To run the commands in shell.in, run
$ ./shell2.pl

To run the commands in a different input file such as shell1.in, run
$ ./shell2.pl shell1.in
